# Personal Workspace — Command Center

A modern, interactive personal workspace for organizing courses, tasks, and notes. Built as a single-page HTML application with no dependencies.

## Features

- **Dashboard** — Overview with stats, recent courses, and notes
- **Courses** — All 9 Google Data Analytics Certificate courses with progress tracking
- **Course 1 Timeline** — Detailed module-by-module breakdown with manual checkbox tracking, segmented category bars, and per-category counts
- **Tasks** — Kanban board (To Do / In Progress / Done)
- **Notes** — Organized notes with color-coded categories
- **Quick Add** — Floating action button to add notes or tasks from anywhere
- **Settings** — Theme (dark/light), accent color, display name, timezone, study pace

## Getting Started

1. Clone the repo
2. Open `index.html` in any browser
3. Or deploy to GitHub Pages for online access

## GitHub Pages

Enable Pages in your repo settings, point to `main` branch root.  
Your site will be live at: `https://YOUR_USERNAME.github.io/workspace/`

## Tech Stack

- Pure HTML / CSS / JavaScript — no frameworks, no build step
- Fonts: DM Sans, JetBrains Mono, Instrument Serif (Google Fonts)
- Progress saved to localStorage
